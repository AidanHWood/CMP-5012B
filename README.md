## Getting Started

Read the [student guide](student-guide.md) for more detailed information on how to use this template.

1. **Clone this repo** (or use it as a template on GitHub).
2. Fill in the templates in `/docs`, `/design` and `/planning` as you progress through the labs.
3. Keep everything under version control:
   - Commit early, commit often.
   - Use branches for features where possible.
4. Use **GitHub Projects or Trello** for your task board.
