## 1.4 Feature Matrix of Similar Systems

| Feature | MyFitnessPal | Strava | Fitbit | Couch to 5K | Apple Health | Health Tracker (Proposed) |
|----------|--------------|--------|--------|--------------|--------------|----------------------------|
| User account / login | Yes | Yes | Yes | Yes | Yes | Yes |
| Record exercise sessions | Limited | Yes | Yes | Guided only | Yes | Yes |
| Record diet / calories | Yes (Strong) | No | Limited | No | Limited (Manual input) | Yes |
| Weight tracking | Yes | No | Yes | No | Yes | Yes |
| Goal setting | Yes | Yes | Yes | Programme-based | Limited | Yes |
| Goal deadline tracking | Limited | No | Limited | Yes | No | Yes |
| Historical trend graphs | Yes | Yes | Yes | Limited | Yes (Strong visualisation) | Yes |
| Group / social features | Limited | Strong | Moderate | No | Minimal | Yes (Opt-in groups) |
| Email notifications | Limited | Yes | Yes | No | Limited | Yes |
| Wearable device dependency | No | No | Yes (Device-based) | No | Partial (Apple devices integrate best) | No |
| Subscription required for full features | Yes (Premium) | Yes (Premium) | Yes (Device + Premium) | No | No | No |


The comparison shows that most existing platforms focus strongly on one area of health tracking rather than providing a fully balanced solution. MyFitnessPal offers detailed dietary tracking but places emphasis on calorie metrics and premium features. Strava focuses mainly on performance tracking and competitive social elements, which may not suit all users. Fitbit provides broader monitoring but is closely linked to proprietary devices. Apple Health effectively consolidates data from multiple sources and offers strong visualisation, however it functions primarily as a data display platform rather than actively supporting behaviour change. Couch to 5K provides structured guidance but is limited to running programmes. There is a clear gap for a simplified system that combines exercise, diet, weight tracking and structured goal monitoring in one place without hardware dependency or subscription barriers. The proposed Health Tracker addresses this by integrating core tracking features with clear goal management and optional group functionality, while remaining accessible and easy to use.
