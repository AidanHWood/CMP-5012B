
## 1.4 Feature Matrix of Similar Systems

| Feature | MyFitnessPal | Strava | Fitbit | Couch to 5K | Health Tracker (Proposed) |
|----------|--------------|--------|--------|--------------|----------------------------|
| User account / login | Yes | Yes | Yes | Yes | Yes |
| Record exercise sessions | Limited | Yes | Yes | Guided only | Yes |
| Record diet / calories | Yes (Strong) | No | Limited | No | Yes |
| Weight tracking | Yes | No | Yes | No | Yes |
| Goal setting | Yes | Yes | Yes | Programme-based | Yes |
| Goal deadline tracking | Limited | No | Limited | Yes | Yes |
| Historical trend graphs | Yes | Yes | Yes | Limited | Yes |
| Group / social features | Limited | Strong | Moderate | No | Yes (Opt-in groups) |
| Email notifications | Limited | Yes | Yes | No | Yes |
| Wearable device dependency | No | No | Yes (Device-based) | No | No |
| Subscription required for full features | Yes (Premium) | Yes (Premium) | Yes (Device + Premium) | No | No |



The comparison shows that most existing platforms focus strongly on one area of health tracking rather than providing a balanced solution. MyFitnessPal offers detailed dietary tracking but places emphasis on calorie metrics and premium features. Strava focuses mainly on performance tracking and competitive social elements, which may not appeal to all users. Fitbit provides wider health monitoring but depends heavily on proprietary devices. Couch to 5K offers structured guidance but is limited to running programmes.
There is a clear gap for a simplified system that combines exercise, diet, weight tracking and goal monitoring in one place without hardware dependency or subscription barriers. The proposed Health Tracker addresses this by integrating core tracking features with clear goal management and optional group functionality, while remaining accessible and easy to use.
