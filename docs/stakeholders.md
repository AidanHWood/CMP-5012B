# Stakeholder List



Stakeholder Groups
- Users
- System Support
- Constultants
- Developers
- Government Bodies


| Group:   Users        | Role / Type             | Needs / Concerns                                    | Typical Job Responsibilities |
|--------------------|-------------------------|-----------------------------------------------------|------------------------------------------|
| Primary end user   | Individuals tracking their health data | Easy navigation, quick data entryb, real-time updates, clear visualisations, data privacy, mobile accessibility| Imput personal health data, monitor progress, review trends, manage goals |
| Healthcare Professionals | Doctors, Nurses, Clinicians| Accurate patient data, reliable reports, secure access, integration within existing systems| Review patient data, monitor conditions, generate reports, support diagnosis and treatment decisions |
| Fitness Enthusiasts | Preventative health users | Performance tracking, potential wearable integration, goal setting, trend analysis| Track fitness metrics, set goals, analyse performance data |

| Group:  System Support    | Role / Type             | Needs / Concerns                                    | Typical Job Responsibilities
|--------------------|-------------------------|-----------------------------------------------------|-------------------------------------------|
| IT Support Staff | Helpdesk, Technical support | Clear documentation, error logs, diagnostic tools | Resolve Technical issues, assist users, manage system querires, troubleshoot problems |
| Cybersecurity Team | Security monitoring | Data protection, access control, vulmerability management | Monitor system security, manage authentication perform vulnerability scans, respond to incidents |
| Database Administrators | Data Management | Data integrity, backup systems, performance optimisation | Manage databases, perform backups, ensure data accuracy, pptimise system performance |
| Human Resources | Organisational wellbeing oversight | Access to reports, GDPR compliance, employee trust, payroll, ethical data handling | Monitor aggregated wellbeing trends, manage consent policies, support health initiatives, ensure legal compliance | 


| Group:   Consultants     | Role / Type             | Needs / Concerns                                    | Typical Job Responsibilities
|--------------------|-------------------------|-----------------------------------------------------|-------------------------------------------|
| Business Consultants | Strategy and Operations advisors | Clear business objectives, measurable KPIs, cost-benefit analysis| Advise on business stratecy, evaluate system impact, analyse return on investment |
| IT Consultants | Technical advisors | System architecture clarity, scalability, integration feasibility| Advise on system design, recommend technologies, ensure integration capability|
| Legal Consultants | Compliance advisors | Regulatory compliance, data protection adherence, risk mitigation |Review policies, ensure GDPR compliance, advice on legal risks |
| Risk Consultants | Risk assessment specialists | System security, reliability, audit trails | Conduct risk assessments, identify vulnerabilities, recommend mitigation strategies | 



| Group:   Developers       | Role / Type             | Needs / Concerns                                    | Typical Job Responsibilities |
|--------------------|-------------------------|-----------------------------------------------------|------------------------------------------|
| Front-End Developers | UI/UX Implementation | Clear design requirements, user feedback, modern frameworks| Design and build user interface, improve usability, implement responsive layouts |
| Back-End Developers | Server-Side development| Well definined functional requirements, database structure clarity | Develop APIs, manage system logic, implement authentication and data processing |
| QA Testers | Quality Assurance | Testable requirements, bug tracking systems, reproducable environments | Perform functional testing, log bugs, validate system performance|
| DevOps Engineers | Deployment and CI/CD | Stable build pipelines, automation tools, documentation| Manage deployment pipelines, automate builds monitor system performance |
| Project Managers | Delivery coordination | Realistic deadlines, clear scope, stakeholder communication | Plan timelines, manage team teaks, coordinate stakeholders, track project progress |

|--------------------|-------------------------|-----------------------------------------------------|---------------------------------------------|
