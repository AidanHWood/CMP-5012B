# User Stories

Use the format:

> As a \<user>, I want \<goal> so that \<reason>.

## Story List

1, As the user Kieran he wants to improve his fitness by reducing his snacking and working out more so that he can spend more time with his 7 children.
2. As the user Kieran he wants to be able to build better habits around his diet, by tracking his daily food intake he wants to ensure that he can improve his health without being overwhelmed.
3. As the user Leonard he wants to create fitness content as an influencer so that he can promote self-improvement onto others.
4. As the user Leonard, he wants to continue improving his physique by tracking progress over time so that he can improve his brand image and stay healthy
5. As the user Olivia, she wants to be able to be able compete with her friends with a ranking so that she can stay motivated in the gym 
6. As the user Olivia, she wants to be able to find vetted friends so that she doesn’t feel uncomfortable or scared to train by herself
